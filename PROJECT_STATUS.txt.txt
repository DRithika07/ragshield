# Agentic AI-Based Detection and Mitigation of Memory Poisoning Attacks in RAG Systems

## Project Overview

AI-powered security platform for detecting and mitigating memory poisoning, prompt injection, jailbreak, role hijacking, and related attacks in RAG systems.

## Tech Stack

### Frontend

* React
* Tailwind CSS
* Framer Motion
* Zustand
* Axios

### Backend

* FastAPI
* LangGraph
* Gemini API
* ChromaDB
* Sentence Transformers
* SQLite

### ML

* Scikit-learn / XGBoost
* ChromaDB Vector Search
* Embedding-based Similarity Detection
